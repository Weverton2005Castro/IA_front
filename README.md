# Chat AI

Projeto React migrado para Vite, com Firebase Auth, React Router e TailwindCSS.

## Comandos

```bash
npm run dev
npm run build
npm run preview
```

## Variáveis de ambiente

Crie um arquivo `.env` a partir de `.env.example` quando quiser sobrescrever a configuração Firebase:

```bash
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
VITE_FIREBASE_MEASUREMENT_ID=
```
